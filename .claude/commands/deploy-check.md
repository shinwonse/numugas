# 배포 전 체크리스트

프로덕션 배포 전에 다음 항목들을 확인합니다.

## 자동 체크 항목

다음 명령어들을 순서대로 실행하고 결과를 보고해주세요:

1. **TypeScript 타입 체크**
```bash
pnpm build
```

2. **ESLint 검사**
```bash
pnpm lint
```

3. **환경 변수 확인**
   - `.env` 파일에 필수 변수가 있는지 확인:
     - `NEXT_PUBLIC_SUPABASE_URL`
     - `NEXT_PUBLIC_SUPABASE_ANON_KEY`

## 수동 체크 항목 (확인 요청)

- [ ] 새로운 API 라우트가 있다면 에러 핸들링이 되어 있는가?
- [ ] 하드코딩된 시즌 연도(예: 2026)가 없는가?
- [ ] console.log가 제거되었는가?
- [ ] 이미지 최적화가 되어 있는가? (next/image 사용)

## 결과 요약

체크 결과를 요약해서 보여주세요:
- 빌드 성공/실패
- 린트 에러 개수
- 발견된 이슈 목록
